# MERCURYCLAVE

A tool to decode text which has been encoded multiple times in base64.

### Installation
1. Clone this repository
2. Perform a pip Installation
  ```sh
  $ pip install .
  ```
3. To uninstall simply type:
  ```sh
  $ pip uninstall MERCURYCLAVE
  ```

### Usage
```sh
$ mercuryclave --file <file_name>
```
